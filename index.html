The nerdy details

This page is written and created using the lovely tools and techniques listed below. It's an HTML5 page with CSS3 magic — no javascript is used, not even for the animations, only to prove that you can go far without it. The CSS is preprocessed with Sass + Compass, and is using a grid written with Susy. All code is forged in Atom and Nano, all graphics assembled in Photoshop, and the entire page was developed using a Git workflow on both Mac OS X and Windows 7/10. Finally, it's hosted over at Github Pages with a CC0 License. The mattias.pw/projects/ page is built with and running on Jekyll.

Fonts used are Engagement for headlines, Noto Serif for paragraph text, and Noto Sans for links. Font Awesome is also used for its, as the name suggests, awesome icons.

Head over to my Github repository if you're interested in looking at the code. And please note that while there is no javascript to be found there, I'm rather well acquainted with it.

Oh, and I should also mention that this site is 100% responsive.
