The nerdy details

This page is written and created using the lovely tools and techniques listed below. It's an HTML5 page with CSS3 magic — no javascript is used, not even for the animations. The CSS is preprocessed with Sass + Compass, and is using a grid written with Susy. All code is forged in Atom, all graphics assembled in Photoshop, and the entire page was developed on Mac OS X and Windows 7, and it's hosted over at Github with a CC0 License.

Fonts used are Engagement for headlines, Noto Serif for paragraph text, and Noto Sans for links. Font Awesome is also used for its, as the name suggests, awesome icons.

Oh, and I should also mention that this site is 100% responsive.
